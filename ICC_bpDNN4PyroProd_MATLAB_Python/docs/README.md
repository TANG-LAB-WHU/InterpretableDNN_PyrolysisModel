# Documentation

This directory contains reference documentation for various components of the pyrolysis model training and SHAP analysis project.

## Available Documentation

- [MATLAB References](matlab_references.md) - Documentation for MATLAB functions and toolboxes used in the neural network model
- [SHAP References](shap_references.md) - Documentation for SHAP (SHapley Additive exPlanations) used for feature importance analysis
- [ICC References](icc_references.md) - Documentation for Illinois Campus Cluster (ICC) used for high-performance computing

## Additional Resources

The project also references the following key resources:

1. **MATLAB Neural Network Toolbox**: Used for implementing the deep neural network model for pyrolysis product yield prediction.
   - [Official Documentation](https://www.mathworks.com/help/deeplearning/index.html)

2. **SHAP (SHapley Additive exPlanations)**: Used for explaining model predictions and feature importance.
   - [GitHub Repository](https://github.com/slundberg/shap)
   - [Documentation](https://shap.readthedocs.io/en/latest/)

3. **Illinois Campus Cluster**: High-performance computing environment used for model training and analysis.
   - [Official Website](https://campuscluster.illinois.edu/)
   - [User Guide](https://docs.ncsa.illinois.edu/systems/icc/en/latest/)

## Using This Documentation

These reference documents are designed to provide quick access to relevant documentation for different aspects of the project:

- Use the **MATLAB References** when working with neural network implementation and model training
- Use the **SHAP References** when performing feature importance analysis and interpreting model predictions
- Use the **ICC References** when running jobs on the high-performance computing cluster

Each reference document contains links to official documentation, tutorials, examples, and best practices.
