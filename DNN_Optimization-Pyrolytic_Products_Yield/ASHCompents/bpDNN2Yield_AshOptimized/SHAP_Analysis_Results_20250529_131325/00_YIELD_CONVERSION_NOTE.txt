IMPORTANT NOTE ABOUT YIELD VALUES
==============================

All yield values in this analysis are presented as absolute percentages (0-100%).
The original model might output negative values or values in decimal form (0-1 range).
For consistency and clarity, all yield values have been:
1. Converted to absolute values (removing negative signs)
2. Scaled to percentage form if in decimal format

This conversion ensures all visualizations use a consistent scale and are easier to interpret.

FILE FORMATS
===========

All plots are provided in three formats:
1. PNG - Raster format for screen viewing and web use
2. SVG - Vector format optimized for Adobe Illustrator editing
3. EPS - Vector format for publication-quality graphics

For Adobe Illustrator users:
- SVG files offer the best compatibility and editing capability
- EPS files are optimized with TrueType fonts for better compatibility
- Both vector formats maintain quality at any scale and are suitable for publication
