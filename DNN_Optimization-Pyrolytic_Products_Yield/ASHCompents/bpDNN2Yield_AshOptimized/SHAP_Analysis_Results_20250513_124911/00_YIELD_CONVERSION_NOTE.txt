Important Note on Yield Values
============================

The neural network model may output yield values in different formats:

1. Decimal form (0-1 range): Common in neural networks with sigmoid/softmax outputs
2. Percentage form (0-100 range): Direct percentage representation
3. Negative values: Can occur due to model constraints or training methodology

This analysis script automatically detects and corrects these cases by:
- Converting negative values to positive (absolute values)
- Scaling decimal values (0-1) to percentages (0-100%)
- Maintaining proper units in all visualizations

These adjustments provide more intuitive interpretation of the SHAP plots
while preserving the relative contributions of each feature.

The waterfall plots show absolute yield values with proper percentage
units and improved labeling for better interpretation.
